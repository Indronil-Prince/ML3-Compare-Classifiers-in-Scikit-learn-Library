Instructions to run the program:
-------------------------------------
1. Ensure you have Python installed on your machine.
2. Install required libraries by running 'pip install numpy pandas matplotlib'.
3. Keep the data files in folders "DryBeansDataset" for Dry beans data and "IrisDataset" for Iris data in same directory of the script.
4. Run the Jupyter Notebook, HW3_Notebook.ipynb, to get the outputs.
